# Intellectual-property boundary

This repository contains independently authored code, documentation, schemas, and synthetic fixtures.

It does not contain:

- third-party Excel workbooks or templates;
- screenshots or exports of reference workbooks;
- copied sheet layouts, named ranges, macros, comments, or formatting systems;
- fixtures derived from proprietary templates; or
- hidden workbook binaries or Git LFS objects.

Spreadsheet binaries are ignored by default. Tests use JSON fixtures and may later generate temporary workbooks from original specifications.

Contributors must confirm that submitted material can be distributed under the project licence.
