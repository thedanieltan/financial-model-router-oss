- deterministic model selection;
- readiness and gap analysis;
- machine-readable transformation plans;

- read workbook structure;
- classify sheets and rows;
- preserve formulas and historical cells;
- produce a semantic workbook map.

- copy the source workbook;
- apply additive patches;
- verify formulas and links;
- support rollback;
- reopen and validate the output.
