Open-source financial modelling infrastructure

Financial Model Router OSS

A deterministic financial-modelling router and workbook-generation foundation.

What this is

FMR OSS helps classify financial-modelling jobs, route them to suitable model families, prepare explicit inputs, and generate auditable workbook outputs.

For practitioners

Use FMR locally to structure modelling tasks, prepare inputs, generate workbooks, and review checks.

For developers

The deterministic core runs locally and should not require API keys for core workflows.

Boundary

FMR does not provide accounting, tax, investment, valuation, or lending advice.

Local use

python -m pip install -e .
fmr serve

Open the local URL shown in the terminal.
