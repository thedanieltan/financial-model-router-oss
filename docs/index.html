Financial Model Router OSS

Open-source financial modelling router and workbook-generation foundation.

What this is

FMR OSS is being refocused around practitioner workflows: choose the right model, collect the right inputs, generate a usable workbook, and run checks.

The first practitioner workflow will be SaaS Budget & Forecast Workbook.

For practitioners

The target experience is a local browser app: enter assumptions in forms, generate an Excel workbook, review checks, and export.

For developers

The deterministic core runs locally and should not require API keys for core workflows.

Boundary

FMR does not provide accounting, tax, investment, valuation, or lending advice.

Current test path

Until the hosted demo is ready, practitioners should test from a local checkout.

python -m pip install -e .
fmr serve

Open the local URL shown in the terminal.
